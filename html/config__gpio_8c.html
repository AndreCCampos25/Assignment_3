Detailed Description

Configuração dos botões.

São declaradas e inicializadas as variáveis e estruturas necessárias para a utilização dos botões. São também definidas as funções que possibilitam a geração de eventos quando um botão é pressionado.

◆ get_bind()

get_bind ( void  )

Configura as interrupções que vão ser geradas quando um botão é pressionado.
São assocociadas funções específicas a essas interrupões.

Cada botão é associado a um pino específico da placa, tornando possível a deteção de
eventos por estes gerados (Clicar num botão). É feita, assim, a configuração das
interrupções geradas pelo clique dos botões.

Se alguma interrupação não for corretamente configurada, o programa apresenta uma
mensagem de erro.
